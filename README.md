# Installing

First clone the repository. then:

For NPM users:

```sh
npm install
```

For yarn users:

```sh
yarn
```

# Building

For NPM users:

```sh
npm run build
```

For yarn users:

```sh
yarn build
```

# Tests

For NPM users:

```sh
npm run test
```

For yarn users:

```sh
yarn test
```

# Running

If you want to use the default `input.json` file (of course, you could change the content of input.json and test it with different values):

For NPM users:

```sh
npm run start
```

For yarn users:

```sh
yarn start
```

If you want to give it another JSON file:

```sh
node dist/index.js filename.json
```
